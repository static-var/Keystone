# Keystone Write Module

## Intent
Create or improve prose, structure, messaging, documentation text, or narrative artifacts.

## Load when
The user asks to draft, rewrite, edit, tighten, outline, explain, or adapt written content.

## Allowed mutation
Only the requested text artifacts or files explicitly in scope.

## Must not
Change code behavior, invent facts, or expand scope into design/implementation without handoff.

## May call
`read` for source material; `research` for factual gaps; `review` for critique.

## Handoff
Deliver the written artifact and note any assumptions or source gaps.

## Exit gate
The output matches the requested audience, format, and constraints.

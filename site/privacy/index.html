Privacy Policy

Effective July 10, 2026

What Keystone is

Keystone is an open-source, skills-only plugin for coding agents. It does not operate an MCP server, require a Keystone account, or provide a hosted service that receives your prompts, files, or credentials.

Data handled by the plugin

The Keystone plugin itself does not collect, transmit, sell, or independently store personal data. Its skill instructions run inside the host product you choose. That host may process prompts, repository contents, tool output, and connected-service data under its own terms, privacy policy, configuration, and your instructions.

Some Keystone workflows may guide the host to read or change files, run commands, or interact with third-party services. Those actions occur only through the host's tools and permission model; Keystone does not receive a separate copy of that data.

Website data

The Keystone website stores only a local light-or-dark theme preference in your browser. It does not use advertising cookies or Keystone-operated analytics. The site is hosted by GitHub Pages, so GitHub may process standard request and security logs under the GitHub Privacy Statement.

The website loads typefaces from Google Fonts. Google may receive standard request information such as your IP address, browser details, and the requested font files under the Google Privacy Policy.

Third-party links

The website and plugin documentation link to OpenAI, GitHub, npm, and other developer tools. Their privacy practices apply when you use those services.

Changes and contact

Material changes will be published on this page with a revised effective date. For privacy questions, use the Keystone support page.
